Static:
HTTP Methods Path Description
GET / (Home/Static Page) Path to Static Page detailing what each path is responsible for handling in the routes.
Users:
HTTP Methods Path Description
GET /user Path is responsible for retrieving all users data within the API.
GET /user/:id Path is responsible for retrieving a users data from the API specifically by their userID, specified in the path.
POST /user/add Path is responsible for allowing new user data to be added into the API.
PATCH/PUT /user/update/:id Path is responsible for allowing data about the user to be altered within the API.
DELETE /user/delete/:id Path is responsible for allowing data about the user to be deleted from the API.
